Session¶

Provides classes for interacting with iTerm2 sessions.

class Session(connection, link)¶

Represents an iTerm2 session.

class KeystrokeReader(connection, session_id)¶

An asyncio context manager for reading keystrokes.

Don’t create this yourself. Use Session.get_keystroke_reader() instead. See its docstring for more info.

get()¶

Get the next keystroke.

Returns:An iterm2.api_pb2.KeystrokeNotification.
class ScreenStreamer(connection, session_id, want_contents=True)¶

An asyncio context manager for monitoring the screen contents.

Don’t create this yourself. Use Session.get_screen_streamer() instead. See its docstring for more info.

get()¶

Gets the screen contents, waiting until they change if needed.

Returns:An iterm2.api_pb2.GetBufferResponse.
static active_proxy(connection)¶

Use this to register notifications against the currently active session.

Returns:A proxy for the currently active session.
static all_proxy(connection)¶

Use this to register notifications against all sessions, including those not yet created.

Returns:A proxy for all sessions.
async_activate(select_tab=True, order_window_front=True)¶

Makes the session the active session in its tab.

Parameters:
  • select_tab – Whether the tab this session is in should be selected.
  • order_window_front – Whether the window this session is in should be brought to the front and given keyboard focus.
async_get_buffer_lines(trailing_lines)¶

Fetches the last lines of the session, reaching into history if needed.

Parameters:trailing_lines – The number of lines to fetch.
Returns:The buffer contents, an iterm2.api_pb2.GetBufferResponse
Throws:RPCException if something goes wrong.
async_get_profile()¶

Fetches the profile of this session

Returns:Profile.
Throws:RPCException if something goes wrong.
async_get_prompt()¶

Fetches info about the last prompt in this session.

Returns:iterm2.api_pb2.GetPromptResponse
Throws:RPCException if something goes wrong.
async_get_screen_contents()¶
Returns:The screen contents, an iterm2.api_pb2.GetBufferResponse
Throws:RPCException if something goes wrong.
async_get_variable(name)¶

Fetches a session variable.

See Badges documentation for more information on user-defined variables.

Parameters:name – The variable’s name.
Returns:The variable’s value or empty string if it is undefined.
async_inject(data)¶

Injects data as though it were program output.

Parameters:data – A byte array to inject.
async_read_keystroke()¶

Blocks until a keystroke is received. Returns a KeystrokeNotification.

See also get_keystroke_reader().

Returns:api_pb2.KeystrokeNotification
async_send_text(text)¶

Send text as though the user had typed it.

Parameters:text – The text to send.
async_set_variable(name, value)¶

Sets a user-defined variable in the session.

See Badges documentation for more information on user-defined variables.

Parameters:
  • name – The variable’s name.
  • value – The new value to assign.
async_split_pane(vertical=False, before=False, profile=None)¶

Splits the pane, creating a new session.

Parameters:
  • vertical – Bool. If true, the divider is vertical, else horizontal.
  • before – Bool, whether the new session should be left/above the existing one.
  • profile – The profile name to use. None for the default profile.
Returns:

New iterm.session.Session.
Throws:

SplitPaneException if something goes wrong.
async_wait_for_screen_update()¶

Blocks until the screen contents change.

Returns:iterm2.api_pb2.ScreenUpdateNotification
get_keystroke_reader()¶

Provides a nice interface for observing a sequence of keystrokes.

Returns:

a keystroke reader, a Session.KeystrokeReader.
Example:
async with session.get_keystroke_reader() as reader:
while condition():

handle_keystrokes(reader.get())

Note

Each call to reader.get() returns an array of new keystrokes.

get_screen_streamer(want_contents=True)¶

Provides a nice interface for receiving updates to the screne.

The screen is the mutable part of a session (its last lines, excluding scrollback history).

Returns:A Session.ScreenStreamer.
pretty_str(indent='')¶
Returns:A string describing the session.
session_id¶
Returns:the globally unique identifier for this session.
class InvalidSessionId¶

class Splitter(vertical=False)¶

A container of split pane sessions where the dividers are all aligned the same way.

Variables:vertical – Whether the split pane dividers in this Splitter are vertical or horizontal.
children¶
Returns:This splitter’s children. A list of Session objects.
pretty_str(indent='')¶
Returns:A string describing this splitter. Has newlines.
sessions¶
Returns:All sessions in this splitter and all nested splitters. A list of Session objects.
