Tab¶

Provides a class that represents an iTerm2 tab.

class Tab(connection, tab_id, root)¶

Represents a tab.

async_select(order_window_front=True)¶

Selects this tab.

Parameters:order_window_front – Whether the window this session is in should be brought to the front and given keyboard focus.
current_session¶
Returns:The active iterm2.Session in this tab or None if it could not be determined.
pretty_str(indent='')¶
Returns:A human readable description of the tab and its sessions.
root¶

A tab’s sessions are stored in a tree. This returns the root.

An interior node of the tree is a Splitter. That corresponds to a collection of adjacent sessions with split pane dividers that are all either vertical or horizontal.

Leaf nodes are Sessions.

Returns:An Splitter forming the root of this tab’s session tree.
sessions¶

A tab contains a list of sessions, which are its split panes.

Returns:A list of Session objects belonging to this tab.
tab_id¶

Each tab has a globally unique identifier.

Returns:The tab’s identifier.
